Bookstore Full-Stack Application

A full-stack Bookstore web application built using Spring Boot (backend) and React (Vite) (frontend).


Key Features

•	Browse available books

•	Add/remove books from cart

•	View total cart price

•	Enter user details before checkout

•	Place orders with success/failure handling

•	Modern UI styled using CSS only

⸻

Tech Stack

Frontend: React, Vite, CSS

Backend: Java 21, Spring Boot, Hibernate, Spring MVC, H2 Database

⸻

Project Structure

bookstore-fullstack/

├── bookstore-backend/

└── bookstore-frontend/

⸻

Running the Project

Backend

cd bookstore-backend

./mvnw spring-boot:run

Frontend

cd bookstore-frontend

npm install

npm run dev
